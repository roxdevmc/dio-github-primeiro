Criando um repositório GitHub. 

Importante lista o README FILE
- nesse aquivo é colocado considerações, links úteis, o que for relevante para o projeto, importante dar atenção. Extensão em md (markdown)


No GitBass entrar na pasta que vc preparou para o projeto/repositorio. 

CODIGOS GIT

git clone (link que foi copiado no github)
git status - lista o tudo que foi feito 
git add ou git add . - ele adiciona os arquivos novos na versão local, volta verde ta pronto git
git commit -m "mensagem do comite, importante, descrever"
git push origin main - envia os todos os comites locais para a nuvem (GitHub)



main, master ou origin - na pasta, aparece quando está dentro do repositorio no Bash
